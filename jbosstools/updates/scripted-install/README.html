Scripted Installation

Scripted installation for JBoss Tools removes the need to step through the Eclipse's installation GUI. This enables simplified repeat installations or installations in bulk.

These scripts will install JBoss Tools via commandline into a new or existing Eclipse 4.3 installation.

 
Ant Scripts

 

These scripts use Eclipse's AntRunner along with comandline arguments for source, target, and what to install.

Run ant -f install.xml help or ant -f director.xml help for commandline syntax, or see examples below.

  • install.xml - Ant script to install from an update site zip, such as jbosstools-Update-*.zip & optionally other sites
  • director.xml - Ant script to install from any update site(s); must list all IUs to install

 
Usage Examples

 

Run ant -f install.xml help or ant -f director.xml help for commandline syntax, or see these examples.

  • examples.cmd.txt (Microsoft Windows operating system)
  • examples.sh.txt (OS X or Linux-based operating system)
