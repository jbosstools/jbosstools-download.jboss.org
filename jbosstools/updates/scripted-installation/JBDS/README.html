There are two major ways to install Red Hat JBoss Developer Studio:

a) Using the izpack installer jar, which will install a self-contained product which includes Eclipse:

  * Download the jar
  * `java -jar /path/to/jbdevstudio-*.jar`

b) Using an update site to install into an existing Eclipse you've already installed:

  * Using the Eclipse Marketplace: https://marketplace.eclipse.org/content/red-hat-jboss-developer-studio-kepler
  * Using the JBoss Developer Studio update site: https://devstudio.jboss.com/updates/7.0/

--- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- --- 

In both cases, this can be done interactively as above, or can be scripted for headless or hands-free installation 
(eg., should you want to install JBoss Developer Studio onto multiple computers in parallel).

  To script the izpack installer to create installation(s) headlessly, see:

    * jbds-install.cmd (Windows)
    * jbds-install.sh (Mac or Linux)

  To script installation into an existing Eclipse install, optionally also installing other features from JBoss Central or another site, see:

    * installJBDS.xml (See also ../README.html for more examples and shell script wrappers.)
