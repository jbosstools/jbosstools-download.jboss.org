JBoss Tools Examples

The examples in this folder can be installed from within JBoss Tools or JBoss Developer Studio via Central.
 

To add new examples to this folder, you must:

  1. Open a JIRA
  2. Fork this repo: https://github.com/jbosstools/jbosstools-download.jboss.org
  3. Make changes to your fork in against the master branch of the jbosstools/examples folder
  4. Commit your changes
  5. Submit a pull request to get your fork's changes pushed to the main repo
  6. Wait up to 6 hrs, or if you have access, you can manually kick this job.
 

Looking to download JBoss Tools? See JBoss Tools Downloads. See also Installation methods.
